# Amazon Sales Dashboard Analysis

## Project Overview

This project analyzes Amazon sales data using an interactive dashboard. The objective is to understand sales trends, customer behavior, and product profitability.

## Skills Demonstrated

- Data Cleaning
- Data Visualization
- Business Analytics
- Power BI Dashboard Design
- Data Interpretation

## Project Structure

- AmazonSalesReport.pbix → Power BI Dashboard file
- Amazon Store Sales Data.xlsx → Dataset used for analysis
- Screenshot.png → Dashboard preview
- README.md → Project documentation

## Dashboard Metrics

* Total Sales: 1.57M
* Total Profit: 175.26K
* Orders: 5.9K
* Customers: 773

## Key Insights

* Consumer segment contributes the highest share of customers.
* Cash on Delivery is the most preferred payment method.
* Standard class shipping is used in most orders.
* Copiers, Accessories, and Phones generate the highest profits.

## Tools Used

* Power BI
* Data Visualization
* Business Analytics

## Business Recommendations

* Promote prepaid payment options to reduce COD dependency.
* Increase inventory for high-profit products.
* Plan marketing campaigns around seasonal demand periods.
